## Angular Fastify demo app

![App screenshot](app-screenshot.png)

### How to run

- Clone this repo
- open a terminal, go to fastify (server) folder and run
  - `npm i`
  - `node server.js`
- go to zoostock (angular client) folder and run
  - `npm i`
  - `ng serve`
